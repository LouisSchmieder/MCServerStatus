# MCServerStatus
A service to check the server status from minecraft servers

# Features:
[x] Favicon
[x] Version
[x] Player info
[x] Player heads
[] Motd
[] Better design

# Usage:
```bash
# Clone the repository
git clone https://github.com/LouisSchmieder/MCServerStatus && cd MCServerStatus

#Run the executable
v run .
```
